বিস্কুটেরarray খায় বিড়াল ছানা

array হচ্ছে একটা বিস্কুটের প্যাকেট

এই চ্যাপ্টারের হিজিবিজি মার্কা প্রশ্ন দেখে তোর মাথা ঘুরে অজ্ঞান হওয়ার আগে তিব্বত কদুর তেল দিয়ে নে।

নিজে নিজে কর

প্রত্যেকটা প্রশ্নের উত্তর নিচে কমেন্ট কর। প্রথমে নিজে নিজে চেষ্টা কর। না পারলে অন্যদের কমেন্ট দেখে চেষ্টা কর। সেটাও করতে না পারলে, কেনো পরতেছস না, সেটা কমেন্ট কর।

৬.১:এমন একটা জিনিসের নাম লেখ, যেটার ভিতরে একই রকমের অনেকগুলো জিনিস থাকে। একই রকমের অনেকগুলো জিনিসের সমষ্টি হলে, সেটা একটা array হবে।

৬.২: তোর পরিবারের সব সদস্যদের নাম নিয়ে একটা array ডিক্লেয়ার কর। এই array এর নাম দিবি family ।

৬.৩:তোর পরিবারের array তে কয়জন সদস্য আছে সেটা বের কর। সেই array এর দ্বিতীয় পজিশনে কে আছে সেটাও বের কর।

৬.৪: ধর, তুই কলিকে বিয়ে করে ফেলছিস। এখন কলিকে তোর পরিবারের সদস্যদের array তে যোগ কর।

  • প্রোগ্রামিং প্র্যাকটিস করে রাত পোহালে
  • string এর তালে নাচবে গরু গোয়ালে

  • নানীর if-else বুঝে লাফায় নানা
  • বিস্কুটের array খায় বিড়ালছানা

  • for লুপ ডাকবে সালিশ
