খুব শিগ্রই বাংলায় সহজ করে লেখা হবে

তখন ইমেইলে আপডেট পেতে রেজিস্ট্রেশন করো: এই খানে

হ্যাশ টেবিল প্রোগ্রামিং করার সময় value হিসেবে সবসময় যে একটা ফোন নাম্বার বা যেকোন একটা মান থাকবে তা কিন্তু নয়। অনেক সময় value হিসেবে অনেক বেশি তথ্য জমা রাখা যায়। যেমন ধর, বাংলাদেশের ন্যাশনাল আইডি (জাতীয় পরিচয় পত্র)। এখন ন্যাশনাল আইডি কার্ডের কথা চিন্তা করলে খুব সহজেই মনে করতে পারবি। সবার ন্যাশনাল আইডি কার্ডে একটা আইডি কার্ডের নাম্বার থাকে।এবং এক জনের আইডি নাম্বার আরেকজনের আইডি নাম্বারের সাথে মিলে না। অর্থাৎ আইডি নাম্বার একটা ইউনিক জিনিস। তাই এইটা দিয়ে টান দিলে নাম, বাপের নাম, ঠিকানা সব চলে আসে। যেহেতু আইডি কার্ডের নাম্বার দিয়ে টান দেয়া হয় তাই এইটা হবে key। আর নাম, বাপের নাম, মায়ের নাম, ঠিকানা, ইত্যাদি হবে value। পুরা বাংলাদেশের সবার ন্যাশনাল আইডি কার্ডের ইনফরমেশন সাথে এক জায়গায় রাখার জন্য bagladesh নামে আরেকটা হ্যাশটেবিল ডিক্লেয়ার কর। সেজন্য প্রথমেই var লিখবি। তারপর bangladesh লিখে সমান চিহ্ন দিয়ে দুইটা সেকেন্ড ব্র্যাকেট দিয়ে দে। নিচের মতো করে। 

	var bangladeshis = { };
যেহেতু বাংলাদেশের ন্যাশনাল আইডি কার্ডে value হিসেবে একাধিক তথ্য থাকে। তাই এই একাধিক তথ্যকে একটা অবজেক্টে রেখে দিবি। তারপর সেই অবজেক্টকে value হিসেবে ব্যবহার করে দিবি। সেজন্যই নিচের মতো হাবলু নামে একটা অবজেক্ট ডিক্লেয়ার কর। 

	var hablu = {
	  name: “Hablu Mia”
	  fathername: “abba kablu”
	  motherName: “ammi jaan”
	  address: “vooter goli”
	}
এখন bangladesh নামক হ্যাশ টেবিলে তোর ইনফরেমশন যোগ করার জন্য আগের মতো, bagladesh লিখে থার্ড ব্র্যাকেটের ভিতরে তোর ন্যাশনাল আইডি নম্বর লিখবি। তারপর ডানপাশে hablu অবজেক্ট বসায় দিবি। তাহলে bangladesh নামক হ্যাশ টেবিলের key হবে ন্যাশনাল আইডি নাম্বার আর value হবে তোর সম্পর্কে হাবিজাবি তথ্য যোগ হয়ে যাবে। 

	bangladesh [8192226339153] = hablu;
যেভাবে তোর নাম ঠিকানাসহ হাবিজাবি তথ্য বাংলাদেশ ন্যাশনাল আইডি কার্ডের তথ্যের জায়গায় রাখা হইছে। একইভাবে অন্য সবার তথ্যও যোগ করা যায়। কিছু কিছু প্রোগ্রামিং ল্যাঙ্গুয়েজ যেমন java বা C# এ হ্যাশটেবিল বা ডিকশনারি ডিক্লেয়ার করতে হয় অবজেক্ট বানানোর মতো করে। শুধু তাই নয় হ্যাশটেবিলের key গুলো কি টাইপের হবে। ভ্যালুগুলো কি টাইপের হবে সেটাও বলে দিতে হয়। তবে আপাততঃ এই ক্যাচালের মধ্যে না গিয়ে- বেসিক কনসেপ্ট ক্লিয়ার করে রাখ। পরে যখন দরকার হবে তখন গুগল ভাইয়াকে জিজ্ঞেস করবি। সে সংজ্ঞাসহ উদাহরণ দিয়ে দিবে। সেই উদাহরণ কপি-পেস্ট মেরে দিবি। কাজ হয়ে যাবে।
