Object Oriented Programming

প্রোগ্রামিংয়ের বলদ টু বস বইতে inheritance, polymorphism, encasulation সম্পর্কে প্রাথমিক ধারণা দেয়া হয়েছে। নিচের লিংকগুলাতে এইগুলার কোডিং দেখানো হয়েছে। যদিও বিস্তারিত বর্ননা এখনো যোগ করা হয়নি।

উপরের তিনটা ছাড়াও আরো তিনটা জিনিস- access Identifier, interface, abstract class এর নাম এবং হালকা পাতলা ধারণা দেয়া হয়েছে। ইন্টারভিউতে এইগুলা সম্পর্কে প্রশ্ন করতে পারে। তবে চাকরিতে যোগ দিলে এইগুলা নিজে নিজে প্রাকটিস করে নিলেই হবে।

  • Encapsulation
  • Polymorphism
  • Access Identifier
  • Interface
